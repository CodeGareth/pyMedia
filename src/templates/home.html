To use this tool:

  1. Please select your mode: Youtube or Flickr
  2. If you select Youtube - please select the style you want - browse (select from 21) or watch (insta-watch one video related to your search)
  3. If you select Flickr - please select the number of pictures you wish to see
  4. Type in what you wish to see in the text box e.g Cats do the craziest of things
  5. Hit submit when you're ready

Youtube search will return a selection of 21 items based on your search query, where as Flickr will return up to 3 random pictures.

Need to reset? Hit refresh in your browser
